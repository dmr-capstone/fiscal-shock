class Walls¶

Handles generation of walls in a dungeon and unfortunately some other miscellaneous tasks.

Public Static Functions

static List<Cell> FiscalShock.Procedural.Walls.setWalls(Dungeoneer d)

Main function to generate walls and update pathfinding graphs. The order of functions is important! Some variables are set in certain functions, and other conditions check for the existence of walls under the expectation that they are (or are not) there.

Return

list of cells that may be used for A* pathfinding
