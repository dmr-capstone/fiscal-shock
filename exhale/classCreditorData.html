class CreditorData¶

Creditor objects used to determine the threat level and whether they have been paid recently or not.

Public Functions

CreditorData.CreditorData(bool beenPaid, int baseThreat)

Constructor. Should be called only once for each creditor ID, or bad things will happen.

Parameters

  • beenPaid: whether this creditor should consider themselves as having been paid to start with

  • baseThreat: base threat level

Public Members

bool CreditorData.paid = false

Whether all of the loans lent by this creditor have had a payment made against them recently. Factored into threat accumulation. They’re happy if you make regular payments!

int CreditorData.threatLevel = 0

Current threat level.
