int FeedbackController.numLossesToDisplay = 60

Maximum number of shot loss feedback objects to display at once; also the size of the object pool. Should be high enough for automatic weapons that can fire once per frame, with the game capped at 60 FPS.

int FeedbackController.numEarnsToDisplay = 16

Maximum number of money gain feedback objects to display at once; also the size of the object pool. Should be high enough for most situations. Consider how many enemies the player might defeat at once, or within 2 seconds.
