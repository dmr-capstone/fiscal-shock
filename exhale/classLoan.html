class Loan¶

Represents a single loan.

Public Functions

Loan.Loan(int num, float tot, float rat, LoanType t, float securityDeposit, string creditorId)

Basic constructor for a loan.

Parameters

  • num: next available ID

  • tot: amount of debt this loan adds

  • rat: interest rate

  • t: type of loan

  • securityDeposit: collateral amount on this loan, should be zero if this isn’t a secured loan

  • creditorId: creditor this loan was borrowed from

Property

property Loan::ID

Unique numeric ID of this loan. Must be unique to pay off the correct loan!

property Loan::total

Amount due on this loan. Increases by day with interest.

property Loan::rate

Interest rate on this loan. The rate remains the same, even if the player’s credit rating drops or rises.

property Loan::paid

Whether the player made a payment on this loan recently.

property Loan::type

The type of loan.

property Loan::age

How many days ago this loan was taken out. Factored into credit rating.

property Loan::originalAmount

Original amount due on this loan, with no interest applied.

property Loan::lender

Creditor ID.

property Loan::collateral

Amount of collateral/security deposit. Will be paid back to the player when the full balance of the loan is paid off.

property Loan::inGracePeriod

Whether this loan is in the grace period. Loans in the grace period don’t accumulate interest. Currently, payday loans have no grace period, and other loan types have a 1 day grace period before interest begins accruing.
