float FiscalShock.Procedural.WeaponGenerator.getStrengthModifier()

Specialized formula to get a curve that lets us hit 1 at x = 0, ~2 at x = 5, ~3 at x = 15, ~4 at x = 30, ~5 at x = 65. Found by using a curve fitting tool and adjusting values. https://mycurvefit.com/

Formula: 7 + (-6/(1 + (x/32)^0.9))

Return

modifier value
