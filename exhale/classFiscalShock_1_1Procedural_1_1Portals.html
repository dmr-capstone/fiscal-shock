class Portals¶

Spawns the escape and delve portals as applicable.

Public Static Functions

static void FiscalShock.Procedural.Portals.makeDelvePoint(Dungeoneer d)

Place the delve portal at a randomized spot. This function should be called before makeEscapePoint! There is no check that a portal isn’t spawned on another portal, and any object that already existed on the point is removed.

Parameters
  • d

static void FiscalShock.Procedural.Portals.makeEscapePoint(Dungeoneer d)

Place the escape portal at a randomized spot. This function should be called after makeDelvePoint! If it is not called last, the delve portal could destroy this portal

Parameters
  • d

static int FiscalShock.Procedural.Portals.makePortal(Dungeoneer d, GameObject portal)

Spawns a particular portal. Removes objects that were already on that spot, rather than trying to find a new place.
