class ShopScript : public MonoBehaviour¶

This is the script that runs the shop. Weapons are generated and the shop sells these to the player.

Public Functions

void ShopScript.dismissTutorial()

Dismisses the tutorial and allows normal gameplay to continue.

void ShopScript.generateNewWeapons()

Generates new weapons in the shop. Every slot in the inventory gets a new weapon and all buttons and markers are set to defaults.

void ShopScript.hoverInventory(int slot)

Display the weapon information. Should be attached to an event listener that deals with hovering over the inventory button.

Parameters

  • slot: associated inventory slot

void ShopScript.blurInventory(int slot)

Hide the previously-displayed weapon information. Should be attached to an event listener that deals with moving the cursor off of the inventory button.

Parameters

  • slot: associated inventory slot

void ShopScript.purchaseWeapon(int slot)

Attempts to purchase a weapon and displays dialog based on the result of the attempt.

Parameters

  • slot: associated inventory slot

void ShopScript.BackClick()

Handles exiting the weapon shop by hiding GUI panels and resetting player state.

void ShopScript.setStateTimesEntered(TMP_InputField inp)

Debug function. Update the StateManager.timesEntered value. This value affects weapon generation.

Parameters

  • inp: input field GUI element

void ShopScript.setStateTotalFloors(TMP_InputField inp)

Debug function. Update the StateManager.value.totalFloorsVisited value. This value affects weapon generation.

void ShopScript.Start()

Sets up the shops inventory and shooting script for later use. Generates weapons for the shop’s inventory whenever the player enters the hub.

bool ShopScript.affordableWeaponsDayZero()

With random generation, the player could, in rare cases, be unable to purchase any weapons on day 0 (due to loan caps). Make sure there’s at least one weapon that’s a reasonable price on day 0.

void ShopScript.Update()

Determines if the player is in range of the shop and pressing the interact key. If so, it activates the menu and unlocks the cursor.

readonly int ShopScript.MAX_GUNS = 9

Hard cap on maximum number of guns the player can hold. Based on the inventory GUI having only 9 slots. The inventory GUI must be redone, similar to the loan GUI, to allow more than the hard cap here.
