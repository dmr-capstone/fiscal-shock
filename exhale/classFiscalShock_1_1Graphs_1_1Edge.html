void FiscalShock.Graphs.Edge.connect(Vertex a, Vertex b)

Properly update the neighborhoods and incident edges of the given vertices. This is separate from the Edge constructor so that it’s possible to create edges for temporary use without mucking up the actual vertices involved, since these are all call by reference. Maybe this should just connect the two vertices p and q of this edge…
