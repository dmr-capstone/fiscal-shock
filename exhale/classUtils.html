static bool Utils.loadFromJson(object values, string filename, bool alreadyLoaded = false)

Deserialize json text into an object and overwrite existing fields of the object. Any fields in the object not declared in the json are left as the defaults of the object.

Return

Parameters

  • values: object to load deserialized data into

  • filename: absolute path to file

  • alreadyLoaded: optional variable to prevent reloading of value, should be tracked by the calling instance
