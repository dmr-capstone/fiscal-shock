class SpawnableEnemy : public SpawnableObject¶

Data class to expose additional fields for enemies that can be spawned during procedural generation. This is currently not used and not up to date with all parameters of an enemy. The original intent was that certain dungeon themes could have different base stats for the same enemy. The enemy stat adjustment for difficulty adjustment does not use these values.
