class Creditor : public MonoBehaviour¶

Runs the loan businesses in the hub; allows for payment and acquisition of debt. Also manages the GUI text fields.

Public Functions

void Creditor.addDebt(ValidLoan associatedLoanData)

Attempts to add a loan of the specified amount of money, failure conditions included and will affect the dialog

void Creditor.payDebt()

Pays the selected loan by the specified amount. If the player pays it off, the win condition is triggered.

void Creditor.checkWin()

Determines if the player has won the game. Called after any loan is fully paid off

void Creditor.BackClick()

Turns off the panel and allows full movement/pause control

void Creditor.dismissTutorial()

Disables the tutorial window and unpauses the game.
