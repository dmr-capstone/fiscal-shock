class BulletBehavior : public MonoBehaviour¶

Behavior of a projectile, after it is fired and left to its own devices.

Public Functions

IEnumerator BulletBehavior.timeout()

Disable the bullet after its lifetime expires. Must be called by the firing script explicitly. Only used by the player, as enemy bullets are not currently pooled.

Public Members

float BulletBehavior.damage = 10
float BulletBehavior.bulletSpeed = 80

Property

property BulletBehavior::target

Body of the target that was hit by the raycast.

property BulletBehavior::localizedTarget

Exact point on the body where the target was hit by the raycast.

property BulletBehavior::hitSomething

Whether this bullet has hit something.

property BulletBehavior::seekingTarget

Whether this bullet has a real, live target it’s seeking.

property BulletBehavior::ricochetEnabled

Whether this bullet can ricochet. Should disable for automatic weapons.

Private Functions

void BulletBehavior.OnEnable()

Set the bullet’s trajectory when it’s enabled (fired).

void BulletBehavior.OnDisable()

Clean up current behavior and references when the bullet is disabled. Disabling implies a return to the object pool, or beginning initial setup of the bullet before firing.

void BulletBehavior.OnCollisionEnter(Collision col)

Behavior when the bullet collides with something.

Parameters

  • col: collision information

void BulletBehavior.FixedUpdate()

Physics updates for the bullet. Handles homing behavior and ricochet when applicable.

Private Members

Vector3 BulletBehavior.ricochetDirection

Direction of ricochet for bullets that don’t expire on hit.

float BulletBehavior.timeSinceLastAiming

How long ago did this bullet (if homing) update its trajectory? Used for enemy homing missiles.

float BulletBehavior.homingUpdateRate = 0.8f

Current hard-coded value for how often enemy homing missiles update. Lower update rates make it really hard to dodge them.
