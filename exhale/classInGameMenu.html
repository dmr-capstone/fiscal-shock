class InGameMenu : public MonoBehaviour¶

In-game/pause menu. Controls the settings, including graphics, audio, and mouse sensitivity.

Public Functions

void InGameMenu.PlayClick()

Exits the pause menu and resumes gameplay.

void InGameMenu.OptionsClick()

Button callback to be assigned to the Options button. Displays the options panel.

void InGameMenu.QuitClick()

Button callback to be assigned to the Quit button. Displays the quit confirmation panel.

void InGameMenu.inventoryClick()

Button callback to be assigned to the Inventory button. Displays the inventory panel. Remaining logic for the inventory panel is present in Inventory.cs, not here.

void InGameMenu.RestartClick()

Button callback to be assigned to the Restart button on the quit confirmation panel. Sends the player back to the main menu.

void InGameMenu.QuitAppClick()

Button callback to be assigned to the Quit button on the quit confirmation panel. Closes the game.

void InGameMenu.CancelClick()

Button callback to be assigned to the the Cancel button… somewhere. It’s the same as the below function. To avoid last-minute bugs from deleting a function without reassigning all references, this is left intact.

void InGameMenu.BackClick()

Button callback to be assigned to the Back button (X). Returns the pause menu GUI to the initial “paused” state.

void InGameMenu.GraphicsClick()

Button callback to be assigned to the Graphics Settings button. Displays the graphics options menu.

void InGameMenu.changeQualityLevel(int level)

Update to a new quality preset

Parameters

  • level: Dropdown index, corresponds to a quality level in the array of valid presets

void InGameMenu.resetToDefaults()

Reset all configurable settings to the defaults of the currently selected preset

void InGameMenu.toggleQualityOverrides(bool toggle)

Toggles whether to apply configurable graphics settings or use the quality preset defaults.

Parameters
  • toggle

void InGameMenu.setVsyncCount(int v)

Change vsync count for software vertical sync. A vsyncCount of 0 is required to cap the frame rate, otherwise, the monitor’s refresh rate is the target framerate.

Parameters
  • v

void InGameMenu.setAntialiasing(int aa)

Updates texture anti-aliasing settings. Values should be powers of: 0, 2, 4, 8 are the only sampling resolutions available in Unity.

Parameters
  • aa

void InGameMenu.setPixelLighting(int lights)

Updates number of times light is calculated on a pixel. More lights means better quality lighting, but drastically reduces performance.

Parameters

  • lights: number of lights per pixel

void InGameMenu.setShadowRes(int res)

Updates shadow resolution settings. Hard-coded to use hard shadows except on the highest resolutions. Soft shadows are more graphics-intensive.

Parameters
  • res

void InGameMenu.setShadowDistance(float dist)

Sets the distance at which the camera starts rendering shadows. Should never be higher than the far clip plane of the camera (128). Setting this high is bad news! In dungeons, the player probably can’t see shadows farther than ~50 units due to perspective.

Parameters
  • dist

void InGameMenu.setAnisotropic(int ani)

Sets anisotropic filtering quality. Anisotropic filtering smooths out textures that are “stretched.” Most noticeable on walls that get squished textures sometimes.

Parameters
  • ani

void InGameMenu.setFramerate(int fps)

Not in the options menu. Requires vsyncCount to be 0 to be applied anyway.

Parameters
  • fps

void InGameMenu.changedAnySetting()

If any setting is customized, we should tick the “override” box, because that’s implied by changing a setting.

void InGameMenu.toggleFPS(bool toggle)

Toggles display of FPS in the top right of the screen.

Parameters
  • toggle

void InGameMenu.toggleFullscreen(bool toggle)

Toggles windowed or full screen mode.

Parameters
  • toggle

void InGameMenu.setResolution(int i)

Updates the game window resolution. Strange things will happen if the player changes their valid resolutions during play (the dropdown is never updated). This wouldn’t happen unless the player hotplugged a different monitor or graphics card, though. Rare cases.

Parameters
  • i

void InGameMenu.loadAllWidgetsFromCurrentState()

Sets up all the graphics widgets to the proper value based on what’s in the settings file.

void InGameMenu.populateDropdowns()

Populates drop downs. Should only ever happen once!

void InGameMenu.closeGraphics()

Close the graphics panel. Implies going back to the main options panel.
