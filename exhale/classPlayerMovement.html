class PlayerMovement : public MonoBehaviour¶

Translates user input into player movement (jumping and running). References: Player Movement: https://www.youtube.com/watch?v=_QajrabyTJc&amp;t=1s Character Controller documentation: https://docs.unity3d.com/Manual/class-CharacterController.html

Public Functions

void PlayerMovement.OnMovement(InputAction.CallbackContext cont)

Updates the 2D movement vector, based on the new input system event handling. This value is not currently used for ground movement.

Parameters

  • cont: context

void PlayerMovement.OnJump(InputAction.CallbackContext cont)

Updates whether the player is currently jumping, based on the new input system event handling. This value is used to allow the player to jump.

Return

Parameters
  • cont

void PlayerMovement.teleport(Vector3 destination)

Teleports the player to a specific destination. The character controller must be disabled to move the player transform, or else the player will snap back into place instantly.

Return

Parameters

  • destination: world space position to teleport to

Private Functions

void PlayerMovement.Awake()

Assigns the input system configuration to the player’s input controller. The new input system is currently only used for jumping. 2D axis movement with the new system handles slightly differently and was not implemented fully.

Assigns the initial player location value in the hivemind.
