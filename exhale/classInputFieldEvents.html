class InputFieldEvents : public MonoBehaviour¶

Class to handle some input field events for better visual feedback. The public void functions should be attached to GUI events on the TM_InputField objects.

Public Functions

void InputFieldEvents.hidePlaceholder()

Hide the placeholder text. Should be called when the input field is focused/selected.

void InputFieldEvents.showPlaceholder()

Replace the current text with the original placeholder text. Should be called when the input field is cleared out.
