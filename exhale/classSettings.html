class Settings¶

Handles control of global game settings and cursor locking/unlocking.

Property

property Settings::cursorStateMutexOwner

Reference to the current owner of the cursor state mutex

property Settings::volume

Passthrough accesses of Settings.volume to the backing data class

property Settings::mouseSensitivity

Passthrough accesses of Settings.mouseSensitivity to the backing data class

Public Static Functions

static void Settings.saveSettings()

Serializes settings data class to JSON and write to file.

static void Settings.loadSettings()

Load settings data from JSON.

static void Settings.updateCurrentSettings()

Updates all settings based on the current values of the backing data.

static void Settings.resetToCurrentQualityDefaults()

Reset configurable graphics settings to the quality defaults.

static void Settings.quitToDesktop()

Save settings to file and quit to desktop. Should be used whenever something needs to close the game.

static void Settings.quitToMainMenu()

Cleans up the state manager and singletons. Should be used whenever something wants to return to the main menu for a blank slate, without closing the game entirely.

static void Settings.forceUnlockCursorState()

Violently unlocks the cursor state and evicts the existing mutex owner, if it was owned.

Do not use except when you NEED to impolitely alter controls, i.e., changing scenes.
