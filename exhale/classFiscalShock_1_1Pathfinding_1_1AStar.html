class AStar¶

The AStar class instance stores an instance of the navigation graph and allows the user to find a path to some destination using the graph.

Public Functions

Instantiates AStar using the given Delaunay Triangulation.

Parameters

Stack<Vertex> FiscalShock.Pathfinding.AStar.findPath(Vertex start, Vertex destination)

Method used to determine the shortest path to the destination point from the start point using the A* algorithm.

Return

Stack containing the vertices that create the shortest path, with the top as the start point.

Parameters

  • start: The point at which to start the pathfinding.

  • destination: The target point of the pathfinding.

Private Functions

void FiscalShock.Pathfinding.AStar.sortedAdd(LinkedList< VertexNode > list, VertexNode node)

Method that takes a node and inserts it into the given list in greatest to least order. Insertion inserts in order of greatest f-cost first, then greatest h-cost as a tie breaker.

Parameters

  • list: List into which to insert the node.

  • node: The node to insert into the list.
