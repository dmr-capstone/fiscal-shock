class EnemyHealth : public MonoBehaviour¶

Manages AI health and reactions to being damaged.

Public Functions

void EnemyHealth.stun(float duration)

Fire off the stun routine if this enemy is not already dead. Should be called after taknig damage to prevent stunned corpses.

Parameters

  • duration: time to remain stunned for

void EnemyHealth.takeDamage(float damage, int paybackMultiplier = 0)

Processes this enemy being struck by something. If the enemy is defeated, the player receives money, a death animation plays, and this enemy is destroyed. If the enemy was not active (enemy is ambushed), the enemy will call for help from nearby enemies.

Parameters

  • damage: amount of damage received

  • paybackMultiplier: modifier on cash earned from defeating this enemy; zero if the player should not gain more than the base value; currently, this is only the case when the enemy is defeated by an exploding object

void EnemyHealth.showDamageExplosion(Queue< GameObject > queue, float volumeMultiplier = 0.65f)

Play particle effect on being struck by pulling it from the object pool. Bigger explosions are louder.

Parameters

  • queue: explosion queue to pull from

  • volumeMultiplier: modifier on audio feedback

void EnemyHealth.addImpact(Vector3 direction, float force)

Creates a knockback impact force on this bot. Currently, the impact force’s effect is only simulated while stunned.

Parameters

  • direction: direction to be blown away to

  • force: amount of force applied
